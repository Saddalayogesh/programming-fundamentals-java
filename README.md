## 📖 About the Repository

This repository contains my Java practice programs focused on **Programming Fundamentals**. It was created to strengthen my understanding of Java basics, improve logical thinking, and develop problem-solving skills through consistent practice.

Whether you're a beginner starting your Java journey or revising the basics, this repository can serve as a valuable learning resource.
